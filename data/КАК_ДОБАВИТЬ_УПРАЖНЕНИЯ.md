# Как добавить упражнения

Упражнения хранятся в файле **`exercises.json`** в этой папке.

## Формат одного упражнения

```json
{
  "id": "ex-5",
  "name": "Название упражнения",
  "description": "Описание: как выполнять, на что обратить внимание.",
  "link": "https://t.me/ваш_канал/123",
  "keywords": ["слово1", "слово2", "синоним", "как ищут"]
}
```

| Поле | Обязательно | Описание |
|------|-------------|----------|
| **id** | да | Уникальный идентификатор (латиница и цифры), например `ex-5`, `ex-10` |
| **name** | да | Название — по нему в первую очередь ищут |
| **description** | нет | Текст с описанием упражнения |
| **link** | нет | Ссылка на пост в канале или видео (можно оставить `""`) |
| **keywords** | нет | Массив слов, по которым тоже будет находиться упражнение |

## Как ищется упражнение

Поиск смотрит:
- **название** (name);
- **ключевые слова** (keywords);
- **описание** (description).

Чем больше подходящих слов в этих полях, тем выше шанс, что упражнение найдётся. Добавляйте в `keywords` синонимы и варианты названий (например: «планка», «планка на предплечьях», «кор»).

## Пример добавления нового упражнения

Откройте `exercises.json` и добавьте объект в массив `exercises` (не забудьте запятую между элементами):

```json
{
  "exercises": [
    ... существующие упражнения ...,
    {
      "id": "ex-5",
      "name": "Берпи",
      "description": "Упражнение: присед → упор лёжа → отжимание → прыжок. Развивает выносливость и координацию.",
      "link": "https://t.me/your_channel/5",
      "keywords": ["берпи", "burpee", "ОФП", "кардио", "выносливость"]
    }
  ]
}
```

Сохраните файл в кодировке **UTF-8**. После перезапуска бота (или при следующем запросе к данным) новые упражнения появятся в поиске.

## Режим SQLite

Если в `config.py` включён режим **SQLite** (`STORAGE_MODE = "sqlite"`), то после изменения `exercises.json` нужно заново заполнить базу:

```bash
python scripts/seed_sqlite_from_json.py
```

После этого перезапустите бота.
